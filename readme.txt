# GJ-LINK
Golang编写性能极高，算法升级（独家），抽水平稳0算力损耗，Web后台实时查看抽水算力，支持本地部署混淆加密客户端，Web后台HTTPS访问。

Web界面操作，简单易用，一键安装，小白可以轻松上手。开机自启动，进程守护运行，自动调整连接数限制。  已支持币种 Eth ETHW ETC ETHF BTC LTC ERGO RVN KAS CFX ZEC DASH DCR 所有币种开发费统一为固定0.2%  程序持续更新中

# 一键安装升级命令

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/ymuako/gjlink/main/install.sh)"
```

> 选择 `1` 为安装服务.
> 选择 `2` 为更新程序.
